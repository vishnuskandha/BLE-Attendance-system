# BLE Attendance System

A smart campus management system for tracking student attendance and irregularities using BLE technology.

## Features

- **Dashboard**: Real-time statistics on student attendance.
- **Attendance Tracking**: View daily attendance records.
- **Irregularities Monitoring**: Track students who are late or skipping classes, with visual evidence.
- **Glassmorphism UI**: Modern, visually appealing interface.
- **Print Friendly**: Optimized styles for printing reports.

## Deployment

This project is designed to be deployed via GitHub Pages.

1.  Push this repository to GitHub.
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch (or `master`) and the root directory `/`.
4.  Save.

## Usage

- **Login**: Use the demo credentials:

    - Password: `admin`
